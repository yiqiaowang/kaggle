To run:

Make sure pytorch, numpy, sklearn are all installed, and the computer has an internet connection (to download the data)

The baseline SVM:
In the Kaggle folder, run:
    python3 SVM.py

The hand-written neural network:
    python3 nn_run.py


The CNN (for our part 3):
Go to https://drive.google.com/file/d/1R2tiljfinSp0VmgqbC2gWrsR5rEaUg4r/view?usp=sharing
This is our Google Colab file. Then click "run all"

Alternatively, run
    python3 cnn.py

Or run the notebook CNN.ipynb either with Jupyter or with Google Colab
